A self-contained static HTML site written for Cottage Aesthetics' owner: the
step-by-step guide that gets her infrastructure fully independent of Josh's fleet,
served by the running admin at `/admin/guide/` and buildable to a standalone folder
she can keep anywhere.

Full specification: `spec/independence/07-html-guide.md`. Contents land in milestone 8
of the independence work plan (`spec/independence/09-work-plan.md`):

- **Start here** — what this guide achieves, the map, total time.
- **Track J** — the handful of steps Josh does once.
- **Track P** — the steps Purdi does, one chapter per sitting (password manager,
  GitHub, DigitalOcean, Cloudflare, Anthropic, the droplet setup, the drill, go-live).
- **Appendices** — if Josh disappears, costs, revoking access.

Nothing here is wired up yet — this file exists so the directory has a place in the
repo ahead of milestone 8's real content (spec/independence/09-work-plan.md, row 8).
